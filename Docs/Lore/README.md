# Lore – Braziliation

This folder holds **world-building and narrative** content for the Brazilian dieselpunk post-apocalyptic setting.

## Purpose

- Establish **tone**, **setting**, and **factions** so art, dialogue, and mechanics feel consistent.
- Give **context for enemies, locations, and items** (why they exist in the world).
- Support **AI agents** when generating descriptions, names, or new content (see AI/Prompts/design_enemy.md).

## Suggested files

- `README.md` (this file) – Overview.
- `world_overview.md` – The world after collapse: regions, power sources, general mood.
- `factions.md` – Major groups, goals, and relation to the player.
- `locations.md` – Key places (industrial, favela, wasteland, etc.).
- `glossary.md` – Terms, slang, and in-world names (Portuguese/BR where relevant).

## Usage

- **Gameplay Engineer** and **Architect** can reference lore when designing enemies and mechanics.
- **Art direction** (AI/Context/art_direction.md) should align with lore tone.
- Update when the narrative or setting evolves.
