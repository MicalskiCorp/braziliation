# Criativo — Gestão Criativa do Braziliation

> Hub de gestão criativa do projeto. Centraliza lendas, narrativa, ideias e brainstorms.
> Alimenta o desenvolvimento de mapas, cenários, monstros e personagens.
> Agente responsável: [`../IA/Agentes/GameCriativoMarkdown.md`](../IA/Agentes/GameCriativoMarkdown.md)

## Navegação

| Seção | Descrição | Arquivo |
|-------|-----------|---------|
| [`Historia/`](Historia/index.md) | Premissa, arcos narrativos e personagens | Historia |
| [`Lendas/`](Lendas/index.md) | Catálogo de lendas brasileiras + mapeamento para jogo | Lendas |
| [`Ideias/`](Ideias/index.md) | Pool de ideias com tags e prioridade | Ideias |
| [`Brainstorm/`](Brainstorm/index.md) | Sessões de brainstorm por tema | Brainstorm |

## Como Usar

| Operação | Comando |
|----------|---------|
| Catalogar nova lenda | `@GameCriativoMarkdown Catalogar lenda: {Nome}` |
| Mapear lenda para elemento | `@GameCriativoMarkdown Mapear: {Lenda} → {monstro\|mapa\|cenário\|NPC}` |
| Registrar ideia | `@GameCriativoMarkdown Registrar ideia: {descrição}` |
| Iniciar brainstorm | `@GameCriativoMarkdown Brainstorm: {tema}` |
| Novo arco narrativo | `@GameCriativoMarkdown Novo arco: {nome}` |
| Novo personagem | `@GameCriativoMarkdown Novo personagem: {nome}` |
| Varredura criativa | `@GameCriativoMarkdown Varredura criativa` |
