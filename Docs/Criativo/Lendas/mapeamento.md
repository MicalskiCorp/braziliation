## Tipos de Elemento

| Tipo | Descrição |
|------|-----------|
| `monstro` | Inimigo ou boss derivado da lenda |
| `cenário` | Ambiente/bioma inspirado na lenda |
| `mapa` | Região navegável do mapa-múndi |
| `NPC` | Personagem não-jogável baseado na lenda |
| `mecânica` | Mechanic de gameplay inspirada na lenda |
| `item` | Item coletável, arma ou equipamento |
