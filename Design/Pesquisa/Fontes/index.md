## Tipos de Fonte

| Código | Tipo |
|--------|------|
| `AC` | Artigo acadêmico / tese |
| `MU` | Museu / instituto cultural |
| `PO` | Portal de cultura estadual / prefeitura |
| `WK` | Wikipedia (ponto de partida; requer confirmação por outra fonte) |
| `LV` | Livro digitalizado / acervo público |
| `JN` | Jornalismo cultural / reportagem |
| `OT` | Outro (descrever) |
